#Лекция №2

"Алгоритмическая система" - набор средств и понятий, позволяющий строить не один алгоритм, а множество алгоритмов, решающих различные задачи.
1. Множество входных объектов
2. СВойства исполнителя алгоритма
3. Множество результатов выполнения алгоритма
4. Язык, на котором формулируются алгоритмы

Два алгоритма являются "эквивалентными" если:
1. Множество исходных данных одного из них являются исходными данными для другого.
2. Применение одного из них к одинаковым исходным данным дает одинаковый результат.

"Математическая модель" - приближенное описание какого-либо класса явлений внешнего мира, выраженное с помощью математической символики.
